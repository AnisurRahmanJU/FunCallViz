# Function Call Visualizer

A **C function call visualizer** that helps learners understand how function calls, recursion, and return values work in C programs. This project visually demonstrates the execution of functions step by step using an interactive graph and code highlighting.

## Features

- Visualize **function calls** and **return values** dynamically.
- Supports multiple example programs:
  - Normal function call (`add`)
  - Factorial (`fact`)
  - Fibonacci (`fib`)
  - Greatest Common Divisor (`gcd`)
- **Step-by-step execution** with a `Next Step` button.
- **Code highlighting** with Prism.js.
- **Interactive graph** of function calls using Vis.js.
- Display **output dynamically** after each step.


## Libraries Used

- **[Prism.js](https://prismjs.com/)** – Syntax highlighting for C code.
- **[Vis.js](https://visjs.org/)** – For interactive, hierarchical function call graph visualization.
- **Graphviz** (optional) – Can be used to generate static function call diagrams from code for offline or PDF purposes.
